### 监督学习与无监督学习的区别？

监督学习使用带标签的数据，无监督学习则不使用标签。
监督学习主要用于预测；无监督学习主要用于寻找数据的内在结构。
监督学习的模型性能通常可以通过准确率、F1分数等指标评估，而无监督学习的评估则更为复杂，通常依赖于领域知识和可视化方法。

### tensorflow 训练的流程是什么？指的是步骤

```
TensorFlow 训练的基本流程通常包括以下步骤：

1. 准备数据
   ⦁ 收集和整理数据集
   ⦁ 划分训练集、验证集和测试集
   ⦁ 数据预处理和标准化

2. 构建模型
   ⦁ 定义模型架构（使用 tf.keras.Sequential 或函数式 API）
   ⦁ 设置层和神经元数量
   ⦁ 选择激活函数

3. 编译模型
   ⦁ 选择优化器（如 Adam, SGD 等）
   ⦁ 定义损失函数
   ⦁ 设置评估指标

4. 训练模型
   ⦁ 使用 fit() 方法开始训练
   ⦁ 设置 batch size 和 epochs
   ⦁ 可选：设置回调函数（如 EarlyStopping, ModelCheckpoint 等）

5. 评估模型
   ⦁ 在验证集上评估模型性能
   ⦁ 使用 evaluate() 方法获取损失和指标

6. 调整超参数（可选）
   ⦁ 根据评估结果调整模型架构、学习率等超参数
   ⦁ 重复步骤 2-5 直到获得满意的结果

7. 在测试集上进行最终评估
   ⦁ 使用未见过的数据评估模型的泛化能力

8. 保存和部署模型
   ⦁ 使用 save() 方法保存模型
   ⦁ 准备模型用于生产环境部署

9. 使用模型进行预测
   ⦁ 加载保存的模型
   ⦁ 使用 predict() 方法进行预测
```
